Welcome to the instructions for the AVsitterPlus™ pose system, as available in Second Life®.

The AVsitterPlus scripts are available here on GitHub and licensed under the Mozilla Public License, Version 2.0.

The AVsitterPlus Documentation covers the following topics:

  • AVsitterPlus instructions.
  • AVsitterPlus contributor guidelines.
  • AVsitterPlus support options.


The AVsitterPlus scripts can be freely obtained from the GitHub repository; however if you would like to receive packaged versions of the latest release, and receive in-world updates of future releases, visit SL Marketplace.

Proceeds are shared with open-source contributors and will help support continued development of AVsitterPlus. Those who purchased AVsitterPlus in the past will continue to receive in-world updates without needing to purchase it again.
